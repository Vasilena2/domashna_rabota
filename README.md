# Проект: Автокъща (Car & Dealer System)

## Описание

Проектът **„Автокъща“** представлява малка C++ система, която моделира работа с автомобили и автокъщи чрез два взаимосвързани класа — `Car` и `Dealer`.  
Той демонстрира прилагането на основни обектно-ориентирани концепции в C++:

- конструктори и деструктори;
- капсулация и достъп чрез getters и setters;
- `const`-коректност и валидация на данни;
- статични членове и методи;
- добри ООП практики в реалистичен контекст.

## Компилация и изпълнение

Отворете терминал в директорията `dealership/` и изпълнете:

```powershell
g++ -std=c++17 -Wall -Wextra -O2 main.cpp -o main
```

След това стартирайте програмата (Windows PowerShell):

```powershell
.\main
```

или в Unix-подобна среда:

```bash
./main
```

## Класове

### Клас `Car`

Представя автомобил с основни характеристики.

**Членове:**
- `std::string brand` — марка  
- `int year` — година на производство  
- `double price` — цена  
- `static int total` — брояч на всички създадени обекти  

**Методи:**
- `Car(std::string b, int y, double p)` — параметризиран конструктор със списък за инициализация  
- `setPrice(double p)` — задава цена с проверка за неотрицателност  
- `to_string()` — връща текстово описание на обекта  
- `static getTotal()` — връща броя на създадените обекти  

---

### Клас `Dealer`

Представя автокъща, която съхранява и управлява наличните автомобили.

**Членове:**
- `std::string name` — име на автокъщата  
- `std::vector<Car> stock` — списък с налични автомобили  

**Методи:**
- `addCar(const Car& c)` — добавя автомобил в наличност  
- `totalValue()` — изчислява общата стойност на всички коли  
- `newerThan(int year)` — връща всички коли, произведени след дадена година  
- `to_string()` — текстово представяне на информацията за автокъщата
